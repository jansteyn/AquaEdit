# AquaEdit
AquaEdit is a simple editor for very large files(>2GB) for 64 bit Windows machines.

Features
--------

- Supports very large files (greater than 2GB)
- 64-bit Windows compatibility
- Simple and intuitive user interface
- Fast and efficient file handling
- Basic text editing functionalities (cut, copy, paste, find, replace)
- Lightweight and minimalistic design
- No installation required (portable application)

Getting Started
----------------
1. Download the latest version of AquaEdit from the [releases page](https://github.com/jansteyn/AquaEdit/releases).
2. Extract the downloaded ZIP file to a desired location on your computer.
3. Run `AquaEdit.exe` from the extracted folder to start the application.
4. Open a large file by clicking on "File" > "Open" and selecting your file.
5. Start editing!
6. Save your changes by clicking on "File" > "Save".
7. Close the application when you're done.
8. For more detailed instructions and tips, refer to the [User Guide](https://github.com/jansteyn/AquaEdit/wiki).
